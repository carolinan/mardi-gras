Mardi Gras is a fun, colorful theme for the carnival season.

== Description ==
Description: Mardi Gras is a fun, colorful blog and event theme that was originally designed for the carnival season. 
The theme has a large header with curved text and a gradient background. It has four widget areas and two menus, including a social menu. 
The default colors are the tradional New Orleans Mardi Gras colors green, purple and yellow. 
Your blog posts are displayed in a 3 column grid and you can change the number of columns and select your favorite colors and fonts in the customizer. 
The theme is responsive and also supports the new block editor with wide and full width blocks.

== Installation ==
	
1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
